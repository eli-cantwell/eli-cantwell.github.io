CSS Positioning Tutorial

To understand the different positions in CSS, we will start off by explaining relative positioning. When an element has a positioning of relative, any positioning adjustment you make to it will be relative to where it originated from. For example:

position: relative;

top: 10px;

An element with this line of CSS will be moved 10 pixels downwards from its original position, with its anchor (where its moved from) being the top part of the element. It’s also important to note that all of these positioning elements (relative, absolute and fixed) do not effect, and are not effected by other elements on the page like they would be when using margin, padding etc.

Now let’s move on to absolute positioning. Any element with the positioning of absolute will look for its closest parent element with the positioning of relative, and any positioning adjustments made will be relative to that parent element. For example:

element-1 {

position: relative;

}

element-2 {

position: absolute;

right: 10px;

}

In this example, element 2 will make its position adjustment of 10px from the right of element 1’s original position.

Lastly, let’s talk about fixed positioning. An element with the positioning of fixed won’t take anything into account, and will be fixed on the page wherever you place it. Positioning adjustments will be relative to the edges of the viewport. For example:

position: fixed;

left: 10px;

top: 10px;

In this example, our element would be placed in the top left corner of our viewport, and will remain there no matter what we do.
