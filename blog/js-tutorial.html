JS Tutorial

If HTML is the bones of a website and CSS is the styling of a website, JavaScript is the code that tells things what to do.

The control flow of a piece of code is the order in which lines of code are executed, and a loop is a piece of code that instructs a your computer to repeat an action until a certain condition is met. Let’s use making a bowl of cereal as an example. The control flow of making a bowl of cereal is quite straight forward. You have a list of steps that you follow chronologically. You add cereal to the bowl, you add milk, then sugar. Now let’s think of eating the cereal as a loop. In this loop, we want to instruct ourselves to repeat the step of eating a spoonful of cereal until we either un out of cereal, or if we are full.

The DOM (Document Object Model) is a way for your JavaScript to take certain things from the HTML of your website so you can then manipulate what it does using JavaScript. For example, in your JS code, you could write:

document.querySelector(‘h1’).innerHTML = ‘Hello’

This grabs the first h1 element from your HTML code and tells it to display ‘Hello’. If we wanted to grab all of the h1 instances in our html we could write:

document.querySelectorAll(‘h1’).innerHTML = ‘Hello’

You can also elect something with a specific Id by typing:

document.querySelector(‘#id-name’).innerHTML = ‘Hello’

Let’s take it a step further and make a button that changes a line of text on our webpage.

First, we need to declare 2 variables, one for the button, and one for the text. In this example, let’s assume that in our HTML we have a button with the id of ‘button-1’ and a paragraph with the id of ‘paragraph-1’. In order to grab these from the HTML and bring them into our JS code, we need to declare the following variables:

let button-1 = document.querySelector(‘button-1’);

let paragraph-1 = document.querySelector(‘paragraph-1’).innerHTML = ‘Old Text’

This grabs our paragraph and our button and assigns them to a variable that we can now manipulate within our javascript. Now we can create an eventListener function to listen out for when the button is clicked, and and bury a function that changes the text of the paragraph inside it. This is how that would look:

button-1.addEventListener(‘click’, function () {

paragraph-1.textContent = ’New Text’;

});

Now when a user clicks on our button the paragraph-1 text will change from ‘Old Text’ to ‘New Text’.

In that last example we used a function to change the paragraph-1 text. Next, we’ll explore what a function actually does. Put simply, a function is a javascript element that performs tasks or calculates a value. Here’s an example:

function simpleAddition(num1, num2) {

return num1 + num2;

}

This declares a function with the name simpleAddition. Now we can call that function by declaring a variable like this:

let answer = simpleAddition(3, 5);

Now if we were to console.log(answer) we would get an output of 8 in our console.
